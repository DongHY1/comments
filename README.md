## Tech Stack

Comments is built on the following stack:

- [React.js](https://react.dev/) – frontend
- [shadcn/ui](https://ui.shadcn.com/) – styles
- [Supabase](https://supabase.com/) - backend
- [Cloudflare](https://cloudflare.com/) – deployments
